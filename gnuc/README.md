# clearn

> study <a href="https://www.gnu.org/software/gnu-c-manual/gnu-c-manual.html">gnuc-c-manual</a>

reference manual for the C programming language as implemented by the GNU Compiler Collection(GCC), this manual aims to document:

- The 1989 ANSI C standard, commonly known as “C89”
- The 1999 ISO C standard, commonly known as “C99”, to the extent that C99 is implemented by GCC

* The current state of GNU extensions to standard C
